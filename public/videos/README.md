# Hero video

Buraya arka plan videosunu `hero.mp4` adıyla koyun:

    public/videos/hero.mp4

- Format: MP4 (H.264), sessiz, döngüye uygun
- Önerilen: dikey/dolgu için ~1080×1920, kısa (10–20 sn) döngü
- İsteğe bağlı poster görseli için `src/app/site-config.ts` içindeki `heroPoster` alanını doldurun.

Video yoksa hero, sinematik bir gradient arka plana zarifçe geri döner.
